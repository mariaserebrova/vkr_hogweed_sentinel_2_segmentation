# Поиск крупных очагов борщевика Сосновского по снимкам Sentinel-2

Репозиторий содержит программную реализацию итогового эксперимента для выпускной квалификационной работы, посвящённой автоматическому выявлению крупных очагов борщевика Сосновского на многоспектральных спутниковых снимках Sentinel-2.

Задача рассматривается как бинарная семантическая сегментация. На вход модели подаются многоканальные фрагменты спутникового растра, на выходе строится карта вероятности принадлежности каждого пикселя классу «борщевик Сосновского».

В итоговой конфигурации использовалась модель U-Net++ с энкодером ResNet34 и десятиканальным входом:

`B2, B3, B4, B5, B6, B7, B8, B8A, B9, B11`.

---

## О репозитории

Итоговый эксперимент изначально выполнялся в Jupyter Notebook.  
В папке `notebooks/` сохранена его исполненная версия со всеми output’ами: графиками обучения, таблицами метрик, полноразмерными картами, результатами объектной оценки и итоговыми визуализациями.

Отдельно код был вынесен в структурированный проект. Это позволяет разделить основные этапы пайплайна:

- загрузку подготовленных многоканальных растров и маски;
- формирование обучающей, валидационной и тестовой выборок;
- обучение модели;
- оценку качества;
- построение полноразмерного прогноза;
- расчёт объектных и буферизованных метрик.

Таким образом, ноутбук сохраняет полный ход финального эксперимента и его результаты, а проектная структура делает программную часть более удобной для просмотра и повторного запуска отдельных этапов.
